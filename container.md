# Neural Stats

This container has common python packages for neural statistical analysis.

**Base Image:** [python3.13.5](https://hub.docker.com/layers/library/python/3.13.5/images/sha256-4ddc936bb22ccb7af9355ed648625db58ca896dd5f40d5fcdbe53b0d7b33ebf5)

## Packages Installed:
Python packages are installed with pip.

### Stats
`numpy`
`pandas`
`scipy`
`scikit-learn`
`scikit-image`
`pingouin`
`opencv-python`
`statsmodels`

### Plotting
`matplotlib`
`plotly`
`seaborn`
`bokeh`

### I/O
`pyarrow`
`h5py`
`tifffile`
`imageio`
`imageio-ffmpeg`
`Pillow`

### Neuro
`isx`
`pynwb`
`pynapple`

### Test
`pytest`
